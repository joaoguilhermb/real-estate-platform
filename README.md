# real-estate-platform
Real estate listing platform built with FlutterFlow and Supabase.
= Projeto de plataforma para listagem de imóveis.

## Tecnologias utilizadas
  - FlutterFlow
  - Supabase

## Funcionalidades
- Listagem de imóveis
- Página de detalhes
- Galeria de imagens
- Filtragem de dados

## Objetivo
Projeto criado para estudo de desenvolvimento de Website ou App em No-Code, e integração com banco de dados.
