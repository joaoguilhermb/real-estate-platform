# Arquitetura do Sistema

A plataforma utiliza uma arquitetura moderna baseada em frontend visual e backend serverless.

## Tecnologias utilizadas

Frontend:
FlutterFlow

Backend:
Supabase

Banco de dados:
PostgreSQL

Integração:
API REST automática do Supabase

## Fluxo do sistema

Usuário acessa o site  >
Frontend solicita dados ao Supabase  >
Supabase consulta o banco de dados  >
Dados são retornados para o frontend  > 
Interface exibe os imóveis
